# [!DNL Catalog Service] Overzicht van de handleiding

Deze handleiding is bedoeld voor beheerders van Adoben [!DNL Commerce]. Het bevat algemene informatie over [!DNL Catalog Service], met inbegrip van aan boord gaan en installeren. Het veronderstelt een basisbegrip van de kern [!DNL Commerce] configuratie en functionaliteit.

[!DNL Catalog Service] heeft twee soorten gebruikers:

* Adobe [!DNL Commerce] systeembeheerders installeren en onderhouden de service. [!DNL Catalog Service] heeft geen Admin UI of eigenschappen voor standaardgebruikers Admin.
* De integrators bouwen GraphQL vragen en integreren hen op een opslagplaats die een frontend cliÃ«nt gebruikt.
